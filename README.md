# PetPal Mobile App

PetPal is a Flutter V1 interactive demo based on the local product proposal and UI reference under `D:\PetPal`.

## Implemented

- Welcome choice flow: upload pet photo or choose from character library.
- Upload-photo demo flow: capture placeholder, photo confirmation, four generated pixel pet options.
- Character library: six preset companion roles.
- Unified pet naming and personality setup.
- Main room: pixel bedroom scene, pet status, feed, clean, caress, and chat mode.
- Profile and settings screens, including widget preview placeholders.
- Local mock state and data models prepared for later Supabase integration.

## Run

Install Flutter, then run:

```powershell
flutter pub get
flutter run
```

If platform folders are missing, initialize them from this directory:

```powershell
flutter create --platforms=ios,android .
```
